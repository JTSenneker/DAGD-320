

From Server
=============

Replication Packet
	
	O	L	TYPE	DESC
	====================
	0	4	ascii	Packet Type (REPL)

	--- Multiple Chunks ---

	?	1	uint8	Object Length (in bytes, including the first 10)
	?	1	uint8	Action(0: create, 1: update, 2: delete)
	?	4	uint32	Network ID
	?	4	ascii	Class ID
	?	?	?		Object Data
